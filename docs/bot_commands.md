# List of all available bot commands

```log
start - login
redeem_code - Redeem code. ex: /redeem_code <actual_code>
balance - get balance data
list_vpn_configs - List vpn configurations available for user
add_vpn_config - Add vpn config to user account
del_config - Delete vpn config from database and from server. ex: /del_config <vpn_config>
del_me - Remove user from database and remove all its vpn configs
me - info about user
add_client - ...
backup_clients - ...
disable_client - ...
enable_client - ...
list_clients - ...
pivpn_user - ...
get_all_users - ...
test_qr - ...
speed_test - Run speed test on vpn server
create_payment - ...
create_invoice - ...
activate_subscription - Activate subscription for customer. Payment will be accepted immediately.
deactivate_subscription - Deactivate subscription for customer.
pay - create invoice
pay_info - get payment informaion
qr - get qr code to vpn config ex: /qr <config_name>
get_config - get vpn config file ex: /get_config <config_name>
```
